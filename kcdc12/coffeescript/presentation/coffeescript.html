What is CoffeeScript?

  • A language
  • An abstraction of JavaScript
  • Compiles down to JavaScript
  • Objeys JavaScript rules
  • Obscures JavaScript flaws

Caffeinated Advantages!

  • Terse syntax
  • Language features borrowed from Ruby, Python, etc.
  • Familiar OOP-style syntax if you want to work with classes
  • Prevents you from creating variables in the global scope
  • Existing JavaScript libraries can be leveraged in CoffeeScript
  • Generated JavaScript can run as fast, or faster, than hand-written code
  • REPL console

Decaffeinated Disdvantages!

  • You need to understand two languages, not just one
  • Runtime errors occur in compiled JavaScript, not CoffeeScript source
  • OOP-style syntax still obeys JavaScript scoping rules (this will mess with your head)
  • CoffeeScript compiler doesnâ€™t generate the best error messages

How do I brew it?

  • Compiled JavaScript can be referenced normally
  • CoffeeScript can be compiled "on the fly" in web pages by including a special script

 


<script type="text/javascript" src="coffee-script.js"></script>
<script type="text/coffeescript" src="my.coffee"></script>
